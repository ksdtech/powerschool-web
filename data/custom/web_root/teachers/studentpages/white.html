Use the navigation pane on the left to work with the students listed.
  • Click on the student's last name to go to your default student screen.
  • Click on the student's first name to go to the last student screen viewed.
    i.e. - If you last looked at an Attendance page, clicking the student's first name will take you to the Attendance page for that student.
  • Click on a class name to change the current list of students to those in that class.

Instructions: To download these email contacts in a CSV file for use with Gmail:
  • IMPORTANT NOTE: You must use Firefox to copy these contacts! Safari will not work! If you are viewing this page in Safari, quit and launch Firefox instead.
     
  • 1. Click Here to pop up a window containing the contact data for these students' parents in a text field.
     
  • 2. Now launch a text editor application, either TextWrangler on a Mac, or Notepad on a Windows PC.
  • 3. Place your mouse cursor inside the pop-up window's text field. Select the entire contents of the text field and copy it to your text editor application's window.
    Hint: You can use Command-A, Command-C to select all and copy, then Command-V to paste.
  • 4. Save the file to your Desktop, making sure to add ".csv" to the end of the file name.
  • 5. After saving the file, you may import it into Gmail's Contacts. See the Learn More link in the Contacts section of Gmail.
